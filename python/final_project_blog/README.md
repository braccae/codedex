# Final Project Blog

### I used gemini instead of openai because it was free

for the most part, the code methods are similar, google requires a different config object and different method calls to generate content.

## To use:

- install dependencies: `uv pip install -r google-genai dotenv`
- copy the .env-EXAMPLE file to .env and add your api key
- run the script: `python main.py`
